# v2ex-comment-markdown

- 让 v2ex 评论 支持 markdown 的渲染, 使用 github 主题
- markdown 内部的代码高亮
- 由于 v2ex 会删除评论代码缩进, 因此插件会自动格式化代码, 支持的语言有 js/jsx/ts/tsx/json/json5/html/vue/scss/css/less/java

## 安装

- github [v2ex-comment-markdown.user.js](https://github.com/lisonge/userscripts/raw/main/packages/v2ex-comment-markdown/dist/v2ex-comment-markdown.user.js)
- greasyfork [449771-v2ex 评论 markdown 支持](https://greasyfork.org/zh-CN/scripts/449771-v2ex%E8%AF%84%E8%AE%BAmarkdown%E6%94%AF%E6%8C%81)
